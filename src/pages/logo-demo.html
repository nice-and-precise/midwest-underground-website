Automatic Theme Switching

The website automatically switches to the white logo in dark mode using CSS filters:

[data-theme="dark"] .logo-image[src*="logo_horizontal_transparent.png"] { filter: brightness(0) invert(1); }

File Formats

  • PNG (1.5MB): High quality, universal support, transparent background - use for main production logo
  • SVG (~650 bytes): Infinitely scalable, tiny file size, crisp at any resolution - use when performance matters
