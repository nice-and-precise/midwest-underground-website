# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and the project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2025-11-23

### Added

- Full Next.js 15 application replacing legacy static site.
- Prisma-based data layer with HDD operational models.
- Authentication via NextAuth v5 with role-based access.
- HDD-focused features: bore logs, daily reports, equipment tracking, 811 tickets, customer management.
- Comprehensive documentation hierarchy under `docs/`.
- Brand standards and naming conventions.
- Serena MCP integration docs and AI context.

### Changed

- README updated to reflect Next.js stack and operations platform.
- CLAUDE.md updated to describe full-stack architecture instead of static HTML.
- Root documentation reduced to core files; all other docs moved under `docs/`.
- Brand palette updated to Charcoal and Safety Orange.

### Fixed

- Broken internal documentation links.
- Mismatched stack descriptions between README, CLAUDE, and code.
- Inconsistent brand colors and typography references.

### Deprecated

- Legacy static HTML documentation now archived under `docs/archive/`.

### Removed

- No docs deleted, but many have been moved under `docs/archive/`.

### Security

- Documented security practices in `SECURITY.md`.
