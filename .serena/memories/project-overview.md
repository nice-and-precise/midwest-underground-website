# Midwest Underground Website - Project Overview

## Purpose
Professional HDD (Horizontal Directional Drilling) field operations management application for Midwest Underground of Minnesota Inc - a directional drilling and underground utilities contractor serving central Minnesota since 1991.

## Current Status
- **Branch:** feat/nextjs-migration
- **Progress:** 40% Complete (Infrastructure ready, features in development)
- **Migration:** From static HTML site to full-stack Next.js application

## Project Goals
1. Replace static website with dynamic application
2. Enable field operations management (bore tracking, daily reports, rod logging)
3. Support role-based access (OWNER, SUPER, CREW)
4. Track HDD projects, bores, daily reports, 811 compliance
5. Provide offline-first capability for field crews
